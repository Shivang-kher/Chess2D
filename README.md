# Chess 2D

Chess 2D is a Chess game made with Unity 2020.3.X version and C# for Android Phones.

## Requirements:
- Unity 2020.3.X
- Git
- Microsoft Visual Studio
- c#
- Chess art (link: https://devilsworkshop.itch.io/pixel-art-chess-asset-pack)
- Phone to export the game
